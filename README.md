# Expense-Management-java
**Java SDK for Expense-Management/**

***
This library contains implementation of Expense-Management SDK in Java. Currently the following [splitwise API](http://dev.splitwise.com/#introduction) calls 
are supported,

`get_current_user`
`get_user/:id`
`update_user/:id`
`get_groups`
`get_group/:id`
`create_group`
`delete_group/:id`
`add_user_to_group`
`remove_user_from_group`
`get_friends`
`get_friend/:id`
`create_friend`
`delete_friend/:id`
`get_expenses`
`get_expense/:id`
`delete_expense/:id`
`get_comments?expense_id=`
`get_notifications`
`get_currencies`
`get_categories`
